# Google-Trending-Vis

**If the live site does not work properly for some unknown reason, follow the steps below**
1. clone this repo
2. `cd api` && `npm i`
3. `npm start`
4. View the project on `https://localhost:9000`

## Available Scripts
### Server side
```
cd api
npm i
npm start
```
### Front end side
```
cd client
npm i
npm start
```
